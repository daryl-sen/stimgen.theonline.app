Colored Blocks Generator

What is Coblo?

Colored blocks is a common type of stimulus used in change blindness experiments. Stimgen helps experimenters generate images containing colored blocks programmatically, drastically reducing preparation times.

Coblo is highly customizeable in terms of each colored block's dimensions, placement, and colors.

Modes

Stimgen Coblo operates in many different modes to suit different experimental needs.

Radial (new!)

Radial mode generates colored blocks which have centerpoints that are equidistant from the fixation cross.

Flip/Change

An image pair is generated each time Coblo runs. One of the pairs will have different colors in the second image (for change blindness). This difference can be a color swap/flip, or a complete change.

Coblo is still being developed as part of the Stimgen v3 update.
